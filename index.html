SQL Injection Explained

1. Basic SQL Injection

Basic SQL injection occurs when user input is directly included in a SQL query without proper sanitization. Attackers can manipulate the input to access unauthorized data.

Example: admin' OR 1=1 --

2. Union-Based SQL Injection

Union-based SQL injection uses the UNION operator to combine the results of two or more SQL queries. It can extract data from different tables in the database.
